\section{Solution}

Using the cosine formula in $\triangle ABC$,
\begin{equation}
    b^2 = a^2 + c^2 - 2ac Cos B    
\end{equation}
\begin{equation}
    (b + c)(b - c) = a^2 - 2ac Cos B
\end{equation}
\begin{equation}
    or, K(b - c) = a^2 - 2ac Cos B
\end{equation}
\begin{equation}
    \text{where }  K = b + c
\end{equation}
\begin{equation}
    Kb + c (2aCosB - K) = a^2
\end{equation}

    Writing (3.4) and (3.5) into matrix form

\begin{equation}
    \begin{pmatrix}
     1 &  1 \\
     K & 2aCosB - K
    \end{pmatrix}
    \begin{pmatrix}
     b \\
     c
    \end{pmatrix}
    =
    \begin{pmatrix}
     K \\
     a^2
    \end{pmatrix}
\end{equation} \\
\vspace{10pt}
Solve matrix (3.6) for 'c' \\
\vspace{10pt}
The coordinates of $\triangle ABC$ can then be expressed as
    \begin{equation}
        \Vec{A} = c
        \begin{pmatrix}
            cos B \\
            sin B
        \end{pmatrix},
        \Vec{B} =
        \begin{pmatrix}
            0 \\
            0
        \end{pmatrix},
        \Vec{C} = 
        \begin{pmatrix}
            a \\
            0
        \end{pmatrix}
    \end{equation}
